永远展现自己最专业的一面

发表于 2019-05-05

在一个职业化的团队里，永远展现自己最专业的一面。

这应该成为一种共识，不然其他人的专业表现会把你的「不专业」衬托的更加一无是处。

因为工作的原因，我逐渐接触到大大小小公司不同职位的工作职位，在这种广泛的接触中，一个有这优秀对接习惯和专业素养的人总能给人留下深刻的印象。

自然，业余、不专业、随意的职业形象也让对这个人，甚至整家公司产生了极差的印象。

不过，我无意声讨别人，更希望可以发现别人的不专业，来修正自己的职业精神。不妨和我一起记录下如何才能永远展现自己最专业的一面？

学习后展现的专业

这一类可能是最简单，最容易学习到的方面了，可能是一个良好的邮件表达，优秀的回复习惯，及时回复的意识等等。总之是可以在简单的学习后就能上手的专业体现，也是学习收益最大的素养。我建议每一个进入大学，开始找工作的同学就开始培养这种习惯。

  • 注册一个正经的邮箱
  • 学习比较严肃的邮箱书写规范和礼仪
  • 好好做一个简历，吐槽下，好多工作多年的同事发的内推简历，我自己都忍不住拒接
  • 简单的问题，自己尝试解决之后再请假别人
  • 不熟悉的人，不要发微信问「在吗？」
  • ······

比较遗憾的是，原本是行业共识的基本问题还是被很多人无视和忽视

时间积累的专业

我印象比较深刻的一句话，来着担任三联生活周刊主编十九年的朱伟，大意是「在杂志社工作十年，你只要不是个傻逼，都可以做到主编。」

时间积累出来的专业性是一眼可以看出来的。

我们可以单纯靠时间熏陶来提升自己的专业，也可以在职场中刻意观察学习，更重要是，要在职场中注意保护自己的「信用」：

但是如果在一家公司工作，同事之间的合作是长期的，一次「没有交代」就会造成伤害，会让与你合作的同事产生「不靠谱」的感觉。如果有两三次「没有交代」，就会让更多人对你不信任。你在这家公司的信任账户其实破产了，这最终对你是一种严重伤害。
— 对人有交代

时间积累出的专业，更多的是一种职场中的「信用」积累，当然「不靠谱」的印象也是会积累的。

「积累」似乎也说明了这个不是那么容易做到的，甚至需要在很多细小的方面来注意调节。那么我就例举个人感觉非常不错的行为，只要长久坚持，这个同事就会给人「靠谱」的印象。

  • 坚持在规定的上班时间前到公司，即便是崇尚「弹性工作制」的互联网公司，形成 「10 点前，我已经开始办公」的确定感
  • 稳定、持续、专注的工作状态，避免过于散漫的工作状态，或者长时间的离开工位
  • 严格执行公司的出差报备、外出同步信息，即便这样显得有点繁琐
  • 在推行 OKR 的公司，注意更新自己的 OKR 进展，并且从内心真正的接受这种工作方式
  • 对待同事热情并保持界限感
  • ······

个人素养中的专业

这部分可能是我认为最为「难以学习」的专业性了，基本属于自己素养和「老天爷给饭吃」，或者称作「天生的」。好多人都因为这一点而发展受限或者迅速成长。

不过，好在，我们还是有机会可以慢慢克制自己向这个方面靠近的。

  • 工作中的「钝感力」，不玻璃心
  • 练习表达，保证清晰、简洁的开会风格
  • 性格干练起来
  • ······

永远展现自己最专业的一面，最重要的前提则是可以不断变得更专业。
而这是一个长期的命题。
